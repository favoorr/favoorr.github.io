小知识点

1.使用整数时，写不写长度是没有区别的，Signed tinyint(1) 和 Signed tinyint(2) 是没有区别的，范围都只能存储到127

2.Signed 和 Unsigned 区别还是很大的

3.能用小点，就小点，占用空间尽量小

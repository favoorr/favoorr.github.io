Vagrant 入门指南

发表于 2017-01-06   |   分类于 programming   |     |   阅读次数

Vagrant 的定位

一个第三方的虚拟机管理软件，平常的虚拟机比如 VirtualBox 是自己下载 iso 文件，
然后建立虚拟机，配置管理环境

使用 vagrant ，它会给你做一些事情，让你更加方便和快捷的使用各种虚拟机，
有个概念要知道，vagrant 本身不是虚拟机，它可以通过 vagrant up --provider virtualbox 显式指定启动的时候使用 VirtualBox 作为虚拟机提供者

为什么要用 vagrant

还有个问题就是为什么要用 vagrant，因为它提供了很方便的配置和使用，相对于自己手工去做，
要方便太多，提供了很多基础的功能，例如虚拟机的建立，启动，销毁，ssh 登录，等等

使用 vagrant 要分清楚的概念

vagrant 新手首先要搞清楚的几个概念，这里简单列举下我的理解方式：

1.首先 vagrant 是一个管理虚拟机的软件，官网地址 https://www.vagrantup.com/downloads.html

2.首先需要安装一个 vagrant 的应用程序，跨平台 Mac Win，都有，官方下载地址 https://www.vagrantup.com/downloads.html

3.vagrant box，这个是 vagrant 官方推出的一种镜像文件，类似之前的 iso 文件，官方搜索地址是 https://atlas.hashicorp.com/boxes/search，比如说想找 centos 的包，或者 ubuntu 的包，都可以来这里找

4.vagrant 安装好之后，可以直接命令行管理，例如 vagrant up 启动一个虚拟机

5.vagrant box 镜像是需要专门管理的，例如 vagrant box list 列出本地所有的包

6.官方有 hello world ，地址在 https://docs.vagrantup.com 可以跟着做，针对国内程序员，要能翻墙，因为 vagrant box 在 Vagrantfile 配置后，是自动下载的，对国内程序员是很不友好的

7.这种工具类的软件知道皮毛最受折磨，后续除了问题会狠狠消耗时间，vagrant 的作者写了一本书 《Creating Development Environments with Vagrant, Second Edition》

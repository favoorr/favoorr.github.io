环境

centos 6.6 64 位

没敢在自己的 mac 上直接撸，怕把 mac 上的 vim 搞残了 …

新手配置vim比较困难，先暂且使用别人的 vim

曾经试着参考别人教程，搞一个属于自己的 vim，可惜 c c++ 基础太弱，一直没实现

正文开始，参考 k-vim readme，和自己跳的坑

1.从 github 克隆到本地

git clone https://github.com/wklken/k-vim.git

2.安装依赖包

# ctags, ag(the_silver_searcher)

2.1 系统依赖

yum install python-devel.x86_64 or yum info python-devel.x86_64

yum groupinstall 'Development Tools'

rpm -Uvh http://download.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-5.noarch.rpm

yum install the_silver_searcher 如果是centos6，需要自己安装，会出现错误 错误：xz compression not available 

需要执行 rm -rf /etc/yum.repos.d/epel* ，使 yum 恢复ok

centos6.6 安装 the_silver_searcher 参考 ，我写的教程

2.2 使用 python

pip install pyflakes
pip install pylint
pip install pep8

要安装很多东西 pip ，参考

# 安装jshint和jslint,用于javascript语法检查
# 需要nodejs支持,各个系统安装见文档 https://github.com/joyent/node/wiki/Installing-Node.js-via-package-manager

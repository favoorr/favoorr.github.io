Mac安装Homebrew

Homebrew介绍

类似yum 和 apt，Homebrew是Mac下的软件包管理工具

官方说明文档

http://brew.sh/index_zh-cn.html

Mac下安装命令(完全复制)

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

在命令行运行后，根据提示安装

基础命令介绍

brew doctor 列出系统的建议和警告信息

brew -v 显示版本信息

brew up/update 更新brew软件包，应该是更新自己吧

brew home package 查看软件包首页

brew info package 查看软件包信息

brew deps package 查看依赖

brew search package 查找

brew option package 查看软件包安装选项

brew install package 如果加 -v 参数，表示显示安装编译信息

brew uninstall / remove package 卸载软件包

brew upgrade package 更新某个软件包，如果不加package参数，表示更新所有软件包

eclipse打开含中文properties文件乱码解决

问题：

Java项目里面经常会使用properties文件，如果在里面添加中文注释，项目转移重新checkout后
如果使用Eclipse默认的properties编辑器就会乱码，解决方案就是安装第三方插件，替换自带的
properties编辑器

选择一 property edit

一个日本开源作者编写的插件，原声编辑器增强版，能够打开含有中文的文件不乱码
算是一个轻量级版本，如果不做大型的国际化功能，这个功能足够

下载地址 http://pan.baidu.com/s/1i3Ibp7R

选择二 rbe

一个功能强大的资源文件编辑工具，支持对key排序和分组展示、直接新建对应国家资源文件等

例如:

user.name

user.password

在展示的时候会展示为user节点，然后下面有两个子节点，如果做大型的国际化网站，这个还是很不错的

下载地址 http://pan.baidu.com/s/1hq89RG4

Java compiler level does not match the installed Java project facet

发表于 2014-08-30   |   分类于 eclipse   |     |   阅读次数

eclipse 报错 Java compiler level does not match the version of the installed Java project facet.

错误信息如下：

Description Resource
Path Location
Type
Java compiler level does not match the version of the installed Java project facet.
new-restDemo
Unknown Faceted Project Problem (Java Version Mismatch)

解决方案：

在项目名上右键–>选择属性–>Project Facets –>Java–>Version 1.7

我的问题是导入项目时出现该问题，开发jdk版本 1.7,

报错是因为这里显示的是1.6，调整为1.7就可以

根据自己情况选择合适的版本

linux环境下，vim粘贴时缩进错乱解决

linux环境下，vim粘贴时缩进错乱解决

粘贴前使用命令

:set paste

原来进入编辑模式，状态提示为 

-- INSERT --

设置命令后，进入编辑模式状态提示为

-- INSERT (paste) --

这样子，粘贴的时候，就可以按照原来复制时的缩进

设置VIM开关快捷键

:set pastetoggle=<F11>

参考博客http://blog.csdn.net/yizhou35/article/details/17366339

意思是在输入 i ,进入编辑模式后，如果要取消缩进，就摁F11，然后在粘贴，等于输入:set paste,
粘贴之后，在摁，等于输入:set nopaste

“+p ,这个在一般vim下不好用，必须使用带图形界面的vim才可以

大概介绍是vim有自己的寄存器，系统有剪切板，然后yy复制的时候，进入寄存器了，

:reg可以查看不同寄存器上已经复制的值,”+ 表示系统剪切板，但是一般的vim不支持

google关键字 vim寄存器 系统剪切板

参考：http://w.gdu.me/wiki/vim/vimcopy.html

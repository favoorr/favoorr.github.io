linux环境下，vim粘贴时缩进错乱解决

粘贴前使用命令

:set paste

原来进入编辑模式，状态提示为 

-- INSERT --

设置命令后，进入编辑模式状态提示为

-- INSERT (paste) --

这样子，粘贴的时候，就可以按照原来复制时的缩进

设置VIM开关快捷键

:set pastetoggle=<F11>

参考博客http://blog.csdn.net/yizhou35/article/details/17366339

意思是在输入 i ,进入编辑模式后，如果要取消缩进，就摁F11，然后在粘贴，等于输入:set paste,
粘贴之后，在摁，等于输入:set nopaste

“+p ,这个在一般vim下不好用，必须使用带图形界面的vim才可以

大概介绍是vim有自己的寄存器，系统有剪切板，然后yy复制的时候，进入寄存器了，

:reg可以查看不同寄存器上已经复制的值,”+ 表示系统剪切板，但是一般的vim不支持

google关键字 vim寄存器 系统剪切板

参考：http://w.gdu.me/wiki/vim/vimcopy.html

ubuntu 13.04 使用笔记

发表于 2014-01-29   |   分类于 ubuntu   |     |   阅读次数

ubuntu 13.04 使用笔记

1.windows和linux双系统

今天自己在家里摸索了下，在win7里面安装了ubuntu linux ，虽然也碰到问题，但还好，
碰到了不少问题，如ctrl+alt+f1 起来的命令行乱码，翻墙代理出错，安装了的软件不知道
怎么运行等，一直到最后翻墙配置成功，算是完成了。

2.ubuntu linux的基本认识

对于linux，到目前为止，使用上基本操作都还好，高深的不会，买了本《鸟哥的linux私房菜》
但是看得不多，没有通篇看，心血来潮吧算是，就想玩下双系统，查了下资料，现在由于
社区强大的力量，已经是傻瓜式安装了，分为两种

第一种：仅仅安装在windows的某个盘，比如说F盘，我当时把这个盘给ubuntu了，然后下载
镜像，用DAEMON 挂载后，安装

第二种：整个电脑换为linux系统，这个代价有点大，而且一般来说，没有已经基础不要这么干，
用了一下午的ubuntu，总体感觉上，还不错，但是如果真要这样子装，我建议自己提前想清楚;)

入门讲解 http://terrychen.info/ubuntu-linux-beginners-guide-preface/

3.ubuntu命令行乱码：

参考上面博客安装好后，图形界面下，没问题，之前在虚拟机下玩过，没遇到太多困难，就是下载的
时候，包比较大，然后云就是很不错的了，可以先下载，然后取回本地，各种云都可以，百度，360 随意
然后就是自己点下各个地方试试，这里说下命令行乱码问题 学会了2个快捷键：

c 指 ctrl ,a 指alt

1.c+a+f1 ~ f6 ，打开命令行ubuntu的命令行窗口，但是可能碰到乱码，花了2小时没搞定，
最后放弃了,c+a+t 出来的窗口，可以代替

2.c+a+t ，打开终端模拟器，也相当于一个命令行窗口，而且这个不会乱码，

4.linux上安装chrome等程序

默认包里面，很多程序都已经有了，比如说听歌的，看视频的，收邮件的等等，默认浏览器
是firefox，我是双浏览器用户，被firefox的同步功能真心伤到，所以平时用chrome，
开发用firefox，开发的各种强大和友好不容置疑，然后就是安装和使用，下面做下笔记：

1.linux上软件包的分类
.tar.gz 源码，要自己编译，然后安装
.rpm redhat 的软件包后缀，可直接安装
.deb debian ,ubuntu 的安装包，可直接安装

2.使用 apt 下载安装 .deb 结尾的包
apt-get install xxx.deb ,只解决获取并安装包，详细用法请自行google
apt-get remove xxx.deb,卸载包
apt-cache search xxxxx 查询有多少个，例如查询 jre 的命令为 apt-cache search
参考链接

http://wiki.ubuntu.org.cn/Apt-get%E4%BD%BF%E7%94%A8%E6%8C%87%E5%8D%97
http://wiki.ubuntu.org.cn/Apt%E5%92%8Cdpkg%E5%BF%AB%E9%80%9F%E5%8F%82%E8%80%83

3.运行程序
安装后如何运行软件，如安装了 google chrome之后，怎么运行哪？
ubuntu里面有个类似与windows下开始的按钮，点击然后输入 chrome，然后就能找到了，运行后，添加到快速启动

查找包 参考博客地址 http://blog.csdn.net/kaiwii/article/details/7327836

5.配置翻墙

用的gae，已经在windows下用了好久，由于不懂python，后来鼓捣了下，也算是好了，firefox 和 chrome成功翻墙成功。

翻墙介绍 http://blog.dimpurr.com/ubuntu-gae/

双系统安装完成。
